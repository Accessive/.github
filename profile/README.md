## 🎮 Current Projects

### Untitled Multiplayer Survival FPS
> *Godot 4 · In Development*

A large-scale multiplayer survival first-person shooter targeting ~150 concurrent players. Built on a server-authoritative netcode architecture using Netfox and ENet. Features component-based design, AOI/interest management, and a full-featured HUD and interaction system.

- Engine: Godot 4
- Netcode: Netfox / ENet (server-authoritative)
- Status: Active development

---

### Godot Group Manager Plugin
> *Godot 4 · Open Source · In Development*

An editor plugin for Godot 4 that streamlines node group management directly within the editor. Built to reduce friction in large scene hierarchies and multiplayer projects.

- Engine: Godot 4 (GDScript / C++)
- Status: Active development

## ♿ Accessibility First

The name says it. Every project we ship is evaluated against accessibility standards from day one — control remapping, colorblind modes, subtitle systems, difficulty options, and beyond. We don't bolt it on at the end.
